# 1. 引言

8年前，三平平无奇高中生设计并开发的，融合众多桌游特色的快乐的文本冒险游戏。

# 2. 基本元素

在一拳一剑的世界里，你将扮演一位武林高手，与一位敌手对决。你的目标即击败对手，为此需要使用计划与行动、从秘籍中学习武功、寻访匠人铸造兵器、以及采取最直接的方式：拳剑相交。

角色的属性值将在后文一一道来。

我们从游戏配件介绍本游戏。

# 2.1 拳剑牌

被称为手牌的拳剑牌(Quanjian card)是游戏的基石。包含了拳，剑，药三种类型。拳与剑作为区分，红色(r)的拳剑为拳，黑色(b)的拳剑为剑。药也具有颜色，但不能作为拳或剑使用。

拳剑在游戏中，用文本Qx_r或Qx_b表示，其中x是拳剑的点数，1.0版本中点数仅仅作为牌的区分，没有实际作用。

TIP

如Q1_r为点数1的拳。Q10_b为点数10的剑。

药比较特殊，点数为J和Q，即有QJ_r、QJ_b、QQ_r、QQ_r四种药。

玩家可以在回合中使用拳/剑进行攻击，受到拳剑的攻击时，需要使用另一种颜色进行防御。具体方法参见后文。

TIP

拳用剑挡，剑用拳挡，互为攻防。红拳，黑剑。

# 2.2 行动牌

行动牌(Action card)作为丰富角色对抗的效果卡出现，与拳剑牌分开计算上限与数量。默认每回合摸1张行动牌，可以选择使用或者不使用。

行动牌分为主动行动牌与响应行动牌。主动行动牌能在主动阶段打出，而响应行动牌不可在主动阶段打出。

WARNING

注意，1.0版本中，如果不小心在主动阶段打出了响应行动牌，行动牌会被系统吞掉。请务必看清发动时机与行动效果。

行动牌具有行动消耗，记为(_)t(_)q，其中t指时间消耗，q指拳剑牌消耗。这里的拳剑牌没有具体说明时，也包括药。

# 2.3 时间系统

每位“武林高手”也需要合理分配时间。游戏有一条时间轴，在角色回合开始时计算用时，每一个行动、普通攻击都需要花费时间。

游戏将每一天分为24小时，7至18为昼时段，19至6为夜时段。游戏默认从早上7点开始。

如果回合结束时的时间在昼时段，那么这个回合则记为昼时段回合。

游戏开始前，可以在程序设定game_mode_setting.yaml文件中修改对应不同时段的回合时间限制。如设定昼时段的回合时间限制为4小时，则角色在每一个昼时段回合的所有行动所花费的时间不能超过4小时。当然，这不是硬性规定，事实上，游戏最大的魅力就在于超时！

若角色回合的使用时间超过了回合使用限制，那么角色将会进入熬夜(sleep late)状态。在熬夜状态下，角色将在下一个自己的回合时，在回合开始前跳过自己的整个回合！

相对的，每一个行动：拳/剑攻击，吃药，绝大部分行动……都需要消耗时间。因此，如何利用时间系统为自己提供优势，需要各位玩家考虑。

TIP

默认情况下，拳与剑消耗1小时。

后续更新将更改这一设定：学霸阵营使用拳时消耗1小时，使用剑时消耗2小时。1.0版本并未实装这一设定。

# 3. 准备开始

# 3.1 挑选角色

1.0版本可由电脑自动选择，也可自行从初设的两个角色中选择一个。具体方法参阅第7章，在程序设定game_mode_setting.yaml文件中修改对应值。

角色选择时，可以从三个角度出发，生命力，技能，阵营。

WARNING

1.0版本中，仅有tyy与jesse两名角色。在无定义的情况下，默认选择jesse进行游戏。

以Jesse为例，Jesse作为一名学霸，拥有5点气血与5点精气。（学霸阵营在1.0版本中无实际意义）

手牌方面：拳剑牌手牌上限为5，行动牌手牌上限为3。

Jesse具有一个先天武功，名为simple is best，可在普通攻击造成伤害时触发效果。

Simple is best：你的主动阶段内，当你的普通攻击造成伤害后，你可以继续使用普通攻击，不计入回合使用限制，时间消耗减1。(没有看起来那么强的一个实用技能)

电脑将自动选择tyy作为英雄。（1.0版本版本中，给了tyy一个打印到控制台的技能。嗯，只能打印一句话……）在程序设定yaml文件中可修改对应值。

# 3.3 气血与精气

每位武林高手拥有气血(hp)与精气(energy)两种衡量生命力的维度。使用精气可以转化为气血，但是游戏中的最普通的药仅仅可以恢复气血值。

当受到伤害时，首先扣除气血值。当气血值已经扣为1时，再扣除气血则需要使用精气转化为气血进行补充。

# 3.4 起始手牌

每位玩家将拥有4张拳剑牌作为自己的初始手牌，每回合在摸牌阶段，双方各摸3张拳剑牌与1张行动牌。

# 4. 进行游戏

# 4.1 回合阶段

游戏按照既定的阶段流程进行。具体分为以下六个阶段：

进入角色1的回合 ->

时间流动阶段 -> 回合开始阶段 -> 摸牌阶段 -> 主动阶段 -> 弃牌阶段 -> 回合结束阶段

进入角色2的回合……

每一个阶段均有各自的行动（注：请无视游戏中出现的判定阶段的字样，1.0版本中没有安排判定区相关，仅有部分临时判定行为。）。

# 4.2 时间流动阶段

回合切换到角色，时间前进1小时。

这1小时计入回合使用限制。

# 4.3 回合开始阶段

作为回合开始，可以触发多种响应行动牌的效果。如果没有行动，则没有需要执行的步骤。可以跳过。

# 4.4 摸牌阶段

每回合在摸牌阶段，双方各摸3张拳剑牌(Q card)与1张行动牌(A card)。

摸到手中的行动卡效果将打印在控制台中。

# 4.5 主动阶段

主动阶段中，游戏提供了所有拳剑牌、行动牌以及结束回合的按钮，输入数字回车则选择某个行动，输入小写字母s则结束主动阶段。

TIP

主动阶段具有诸多限制，如默认情况下，每个角色在主动阶段仅可以使用1张拳剑进攻，称为回合使用限制。但Jesse的技能以及一些行动可以打破该限制。

# 4.6 弃牌阶段

主动阶段结束后，将根据角色的拳剑牌上限和行动牌上限进行弃牌，在弃牌阶段，手牌数超出限制的，应弃牌至与限制数量相等。默认的上限是5张拳剑牌和3张行动牌。

TIP

若该角色的弃牌阶段被跳过，则不需要弃牌。任何阶段都相同。

# 4.7 回合结束阶段

作为回合结束，检查是否熬夜，也作为部分行动结算的阶段出现。

# 5. 角色死亡

当一名角色气血值降为0，且手牌中没有药时，则使用该角色精气进行一比一补充，即消耗1点精气转化为1点气血。

而当一名角色气血降为0，且精气仅为1时，角色进入元神状态。

元神状态下，角色无法被选择为攻击的目标，也无法对元神角色造成任何气血的伤害。

同时，元神状态的角色，在回合开始阶段需进行一次“生死投掷”，若硬币为正面，则正常进入流程，若硬币为反面，则角色死亡。死亡后的角色将被移出游戏。

最后需注意，在弃牌阶段，元神角色需要弃置全部的拳剑牌与行动牌。

# 6. 游戏结束

当有角色死亡时，若剩余玩家数量为1，则游戏结束！为了消灭其他角色努力吧！

# 7. 基本概念补充

# 7.1 游戏配置文件含义

游戏配置文件需与exe放置于同一目录中，名称为game_mode_setting.yaml不可更改。

## 拳剑 用户设置
# 昼夜时间
round_day_time: 4       # 昼回合时间限制
round_night_time: 3     # 夜回合时间限制
# 玩家数量
game_play_num: 2        # 暂不支持多人……可能会有无法预料的bug
# 对手选择 | 'random' 或 [0,1] 
opponent_choice: [1]    # 如果需要jesse当对手，可以输入 [0]
# 英雄选择 | 'random' 或 0 1 2
hero_choice: 0
# 文本滚动间隔
text_sleep_time: 0.3    # 游戏文本的速度间隔，有时会刷新过快导致没观感
